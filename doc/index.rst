pyHTTPInterface Documentation

Installation/Updates
********************

Installation using PIP
======================

.. code-block:: bash

   pip3 install pyHTTPInterface


Updating using PIP
==================

.. code-block:: bash

   pip3 install -U pyHTTPInterface
